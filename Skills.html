2023 – C++

I began learning C++ in COS 132 – Imperative Programming and COS 110 – Program Design. These modules introduced me to the fundamentals of programming, problem solving, and object-oriented design. I gained experience with types, arrays, control structures, classes, inheritance, polymorphism, memory management, and data structures like stacks and queues. Later, in COS 214 – Software Modelling, I learned to use C++ in modelling complex software systems with design patterns and reusable structures. C++ remains a core language for strengthening my logic and programming foundations.

2023 – Python

I was introduced to Python in WTW 152 – Mathematical Modelling. While we didn’t go very deep into Python itself, I used it as a tool for difference equations, curve fitting, and linear programming. Applications were drawn from real-life problems in finance, economics, and ecology. This experience showed me how programming integrates with scientific and mathematical problem solving, and gave me exposure to applying Python in practical, modelling-focused contexts.

2024 – Java

I started learning Java in COS 212 – Data Structures and Algorithms, where I worked with advanced data structures such as trees, heaps, hash tables, and graphs. I also studied sorting, searching, and traversal algorithms, comparing their efficiency and implementation in Java. In COS 226 – Concurrent Systems, I explored the fundamentals of multithreading, synchronization, and program correctness in concurrent environments. Java became the language where I learned how to design efficient, scalable, and concurrent programs.

2024 – Web Development

Through COS 216 – Netcentric Computer Systems, I learned the fundamentals of modern web development. I worked with HTML, CSS, JavaScript, and Python, understanding both client-side and server-side development. The module also introduced concepts of cloud storage, mobile platforms, data synchronization, and virtualization, preparing me for full-stack development. This was my entry into building dynamic, interactive, and responsive web applications.

2024 – Databases

In COS 221 – Introduction to Database Systems, I learned how to model and manage databases conceptually and relationally. We used MariaDB for practical work, exploring SQL queries, relational algebra, functional dependencies, normal forms, indexing, and transaction processing. This gave me practical skills in designing and querying relational databases, as well as understanding the principles behind modern data management systems.
